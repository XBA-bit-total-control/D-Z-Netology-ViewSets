## Выполнение задания по взаимодействию с ViewSet`ами

Для работы проекта нужно

Библиотеки установить:

```bash
pip install -r requirements.txt
```

БД в settings.py настроить и после миграции применить:

```base
python manage.py migrate
```

Сервер запустить:

```bash
python manage.py runserver
```
